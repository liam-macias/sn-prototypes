# sn-prototypes

A shared space for SimpleNursing product prototypes — design explorations, strategy docs, and interactive mockups for internal review and team feedback.

## Purpose

This repo hosts prototype files so they can be easily shared with the team. Nothing here is production code — these are working documents used to communicate ideas and gather feedback before anything is built.
